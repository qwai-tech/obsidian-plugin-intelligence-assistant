# Bilibili 视频脚本提纲

## 标题（二选一）

- **版本 A：** 我把ChatGPT网页版、Perplexity全替换了，就靠这一个Obsidian插件
- **版本 B：** 用Obsidian的朋友有救了！这个AI插件让你永远不用切Tab

## 视频时长建议

6-8 分钟（保持节奏感，避免拖沓）

---

## 分镜脚本

### [0:00 - 0:30] 开场钩子（悬念）

**画面：** 快速切换多个软件窗口 — Obsidian、ChatGPT、Perplexity、另一个 AI 工具

**台词：**
> "你每天用 AI，要同时开几个软件？ChatGPT，Perplexity，还有这个，还有那个…… 光是 Alt+Tab 一天要按多少次？"
> "从今天开始，我只用一个。"

**切入：** 展示 Obsidian 界面，侧边栏打开 Intelligence Assistant 聊天面板

---

### [0:30 - 1:30] 痛点共鸣

**画面：** 屏幕录制演示"过去的工作流"

**流程演示：**
1. 在 Obsidian 写笔记，遇到问题
2. 切到 ChatGPT 网页
3. 手动复制笔记内容粘贴进去
4. 得到答案，再切回 Obsidian 粘贴

**台词：**
> "这就是我之前每天要重复几十次的操作。你有没有觉得这很蠢？AI 明明应该帮我省时间，但光是切窗口、复制粘贴就消耗了大量精力。"

---

### [1:30 - 3:00] 插件介绍

**画面：** 打开 Obsidian 设置 → 社区插件 → 搜索 Intelligence Assistant → 安装

**台词：**
> "Intelligence Assistant 是一个 Obsidian 插件，它做了一件听起来简单但很难做好的事：把完整的 AI 工作站搬进 Obsidian。"

**功能列表逐条展示（画面跟着切换）：**
- 聊天界面 → "多模型对话，OpenAI、Claude、Gemini、DeepSeek，甚至 Ollama 本地模型，在这里一键切换"
- 模型选择器 → "不需要改配置，顶部直接选"
- RAG 开关 → "开启这个，AI 就能读你的笔记了"
- Web 搜索开关 → "开启这个，AI 就能搜网页了"

---

### [3:00 - 5:00] 核心功能演示（高潮段）

**演示 1：RAG**

**台词：**
> "我在 vault 里有几百篇读书笔记。现在我直接问它：'我关于《思考，快与慢》记录了哪些核心观点？'"

**画面：** 输入问题 → AI 回答并附上笔记引用来源

> "它直接从我的笔记里找到了，还标注了来源文件。这才叫真正认识我的知识库。"

**演示 2：Agent 模式**

**台词：**
> "现在来个更复杂的——我让它帮我研究一个话题，搜索最新资料，结合我的笔记，然后写一份总结插进当前文件。"

**画面：** 切换到 Agent 模式 → 输入指令 → 展示 Agent 自动执行步骤：网络搜索 → 读取笔记 → 生成总结 → 插入文件

> "它自己搜索、自己读笔记、自己写总结、自己插入文件。一个指令，全程自动。"

**演示 3：MCP 工具（可选，视受众）**

> "如果你是开发者，还可以接入任意 MCP Server，把外部工具的能力直接带进 Agent 循环。"

---

### [4:30 - 5:00] Quick Actions 演示

**台词：**
> "还有一个功能，我每天用得最多——Quick Actions。不需要打开聊天，直接在笔记里选中一段文字，右键，选'总结'，一秒出结果。还有解释、修语法、改写、扩写，全在右键菜单里。"

**画面：** 在笔记中选中一段文字 → 右键菜单展示 Quick Actions → 点击「总结」→ 结果直接插入文件

> "对写作的人来说，这个功能绝对是用得最多的。选中即用，完全不打断思路。"

---

### [5:00 - 6:00] 安装演示

**画面：** 完整安装流程（录屏）

1. Obsidian 设置 → 社区插件 → 浏览
2. 搜索 "Intelligence Assistant"
3. 安装 → 启用
4. 在设置里填入 API Key
5. 打开聊天面板开始使用

**台词：**
> "整个安装过程不超过两分钟，配置你的 API Key 就可以用了。"
