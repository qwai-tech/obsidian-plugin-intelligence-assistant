## 正文

😫 每次用 AI 都要离开 Obsidian 切到 ChatGPT？复制粘贴来回跑，太烦了。

终于找到了解决方案——**Intelligence Assistant**，一个把 AI 工作站直接塞进 Obsidian 的插件。

核心能力：

🤖 **多模型聊天** — OpenAI / Claude / Gemini / DeepSeek / Ollama 本地模型全支持，一个界面随意切换
📚 **RAG 知识库** — 索引你的 vault，AI 回答直接引用你自己的笔记
🌐 **网络搜索** — 支持 Google / Bing / Brave 等8家，实时信息直接注入对话
⚡ **Agent 模式** — 不只是聊天，能自动调用工具、多步骤完成任务
✨ **Quick Actions** — 选中任意文字，右键即可触发「总结 / 解释 / 修复语法 / 改写 / 扩写」，一秒完成
🔧 **MCP 工具** — 接入任意 MCP Server，能力无限扩展
📡 **OpenAPI 工具** — 把任意 API 变成 Agent 可调用的工具

安装方式超简单：
Obsidian → 设置 → 社区插件 → 搜索 **"Intelligence Assistant"** → 安装启用

💡 支持 Ollama 本地模型，数据完全不出你的电脑
