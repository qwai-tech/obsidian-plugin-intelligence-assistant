# 平台账号申请准备清单

## 国内平台

### 小红书
- [ ] 中国大陆手机号（用于注册 + 验证）
- [ ] 真实姓名（创作者认证需要）
- [ ] 身份证（申请"专业号"需要实名认证）
- [ ] 建议开通「专业号 → 其他 → 科技」分类，获得更多曝光权重
- **注意：** 新账号发布内容初期推荐量有限，建议先完善个人主页再发第一条

### 微信公众号
- [ ] 中国大陆手机号
- [ ] 身份证（个人订阅号实名认证）
- [ ] 邮箱地址（账号注册）
- **选择账号类型：** 个人 → 订阅号（推广内容选这个，不需要营业执照）
- **注意：** 公众号每天只能群发1条，建议先积累几篇文章再开始推广

### 知乎
- [ ] 中国手机号或邮箱均可注册
- [ ] 建议完善个人资料：头像、简介、所在领域（填"互联网/AI/效率工具"）
- [ ] 个人认证可提升回答权重（需要身份证实名）
- **注意：** 新账号发文章审核较严，建议先回答几个相关问题积累信誉

### V2ex
- [ ] 邮箱注册即可（无需手机号）
- **注意：** 新账号有发帖冷却限制，注册后需要先浏览一段时间才能正常发帖
- **技巧：** 发帖选择合适节点，推荐 `/go/programmer` 或 `/go/obsidian`

### Bilibili
- [ ] 中国大陆手机号（注册）
- [ ] 身份证（发视频需实名认证）
- [ ] 创作者中心认证（可提升推荐流量，需要100粉或满足条件）
- **注意：** 投稿视频会经过审核（通常1-2小时），建议避免敏感词
- **技巧：** 上传时填写完整标签和分区（科技 → 数码 → 软件应用）

## 国际平台

### Twitter / X
- [ ] 邮箱或手机号（国际手机号均可）
- [ ] 国内访问需要 VPN
- [ ] 建议注册后先完善 Bio，关注 Obsidian / PKM 社区相关账号
- **技巧：** 发推时 @obsdmd（Obsidian 官方账号）可获得更多曝光

### Reddit
- [ ] 邮箱注册（无需手机号）
- [ ] 国内访问需要 VPN
- **注意：** 新账号 karma 为0，部分子版块要求最低 karma 才能发帖
  - r/ObsidianMD：无严格限制，但新账号发的帖子可能被过滤
  - 建议：注册后先在一些通用版块评论几天，积累一点 karma 再发推广帖
- **注意：** Reddit 对露骨自我推广比较敏感，文案要以"社区分享"为主，避免纯广告语气

### ProductHunt
- [ ] GitHub / Google OAuth 或邮箱注册
- [ ] 国内访问需要 VPN
- **重要：** ProductHunt 发布产品前，建议提前1-2周在社区活跃（给其他产品点赞、评论），获得更高的社区信任度
- **发布时间建议：** 选择太平洋时间（PST）00:01 发布，争取一整天的曝光窗口（通常是周二至周四效果最好）
- **Hunter：** 可以邀请一位有影响力的 ProductHunt 用户作为 Hunter，能显著提升曝光

### GitHub
- [ ] 邮箱注册（已有账号直接用）
- 确保 README 包含：徽章、截图、清晰的安装步骤
- 发布到 Awesome Obsidian 列表可带来持续流量

---

## 发布顺序建议

1. **GitHub** — 最先确保 README 完善（其他平台会链接到这里）
2. **V2ex + Reddit** — 技术社区，先测试反馈
3. **知乎 + ProductHunt** — 深度内容平台，需要准备时间
4. **小红书 + Bilibili** — 需要账号预热，内容准备好后集中发布
5. **Twitter/X + 微信** — 配合其他平台同步扩散
